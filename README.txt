Creation d'un petit Forum :

Ce sit permet de communiquer avec plusieur utilisateurs par le biai d'une 
adresse d'un server mise en ligne .
Le but etaits de créer un petit forum comme reddit ou stackoverflow et 
d implementes les Webservices suivants:

-new_thread(info) Créé un nouveau thread. info est un string qui contient
ce que vous avez à dire.

-get_threads Récupère la liste des threads

-show_thread(id) Récupère le fil de conversation. id est l’id du thread que vous souhaitez récupérer

-reply_to_thread(id, info) Ajoute un message dans un thread. id est l’id
du thread, info est le message à ajouter

-delete_thread(id) Supprime un thread. id est l’id du thread

Le lien racine des web services etait http://tp-iwa.waxo.org/
